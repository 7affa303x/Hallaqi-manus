## RC Polish (10-point audit)
1. UX Audit — spacing, alignment, overflow, clipping
2. Navigation Audit — flows, back buttons, hidden actions
3. Responsive Audit — all breakpoints
4. Interaction Polish — buttons, dialogs, forms, animations
5. Feature Completion — finish or replace placeholders
6. Consistency Audit — unified design language
7. Performance — remove dead code, optimize
8. Accessibility — ARIA, keyboard, contrast, touch targets
9. Production Cleanup — remove debug code, logs, unused files
10. Final Review — full QA pass
